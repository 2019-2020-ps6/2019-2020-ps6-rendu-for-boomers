# Installation
### ouvrir une console
```
./install.sh
```
# Lancement
### ouvrir une console dans le dossier back-end
```
npm run dev
```
### ouvrir une console dans le dossier front-end
```
ng serve
```

# Required Packages

### npm 

```
npm install
```

### angular

```
npm install -g @angular/cli
```

### ng2-charts

```
npm install ng2-charts
```

### chart.js

```
npm install chart.js
```

### run front

```
ng serve
```
### run server

```
npm run dev
```
